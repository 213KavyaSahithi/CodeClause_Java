This Java project is an Simple Hotel Booking System that simulates the basic functionalities of an online reservation platform. It includes features such as login,logout,check-in,check-out,room status

Here's a summary of the main components and features:

Login: Enter username and password to log in. Invalid credentials will result in a login error message.

Credentials to log in:

Username: admin

Password: admin123

Check-In: After logging in, you can check-in guests to vacant rooms. Occupied rooms cannot be checked in.

Check-Out: Check out guests from occupied rooms. Vacant rooms cannot be checked out.

View Room Status: View the status of all rooms (vacant/occupied) after logging in.

Logout: Click the "Logout" button to log out of the system.
